# Next.js / React Lernportfolio (Kurzueberblick)

## Ziel
Kompakter Lernpfad fuer Frontend bis Full-Stack Basics mit Next.js, TypeScript und React.

## Enthaltene Kursquellen
- `00_codecademy_intro`: Next.js Intro (App Router, TS, Tailwind)
- `00_udemy_nextjs`: mehrere Lernstaende (Next v12/v16 + kleine React App)
- `00_webdevsimplified_react`: Zusatzbereich React (aktuell leer/Platzhalter)
- `01_ai-tutor_theorie`: 13 strukturierte Theorie-Tage mit kommentiertem Beispielcode
- `02_ai-tutor_uebungen`: Tutor-Aufgaben inkl. `loesung_1` / `loesung_2`
- `03_mike_uebungen`: eigenstaendige Uebungen je Modul
- `04_mike_next_abschlussprojekt`: Abschlussprojekt-Planung und Nachweis

## Kernkompetenzen (HR Keywords)
Next.js, React, TypeScript, App Router, Routing, Layouts, API Route Handlers, Server Actions, Caching/Revalidate, Testing Basics, Deployment Vorbereitung, Tailwind, Bootstrap.

## Nachweisstruktur
Theorie -> AI-Tutor-Uebung -> Eigenleistung -> Abschlussprojekt.
